Legend List is a fast and simple list library for React Native. Compared to FlatList and FlashList it's faster, simpler, and has more features.

- ✨ Extremely fast
- ✨ 6KB of 100% JS
- ✨ Dynamic item sizes
- ✨ Optional recycling
- ✨ Recycling hooks
- ✨ Chat list without inverting
